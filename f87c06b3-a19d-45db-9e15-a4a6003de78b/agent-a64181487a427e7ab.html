### Git integration

Set up automatic transcript capture when a session ends:

```sh
cg install --agent claude --format html

This creates a .transcripts/ directory, installs a SessionEnd hook that renders transcripts on session end, and adds .transcripts/ to .gitignore.

To control the output directory name:

cg install --agent claude --format html --out .transcripts

To enable orphan-branch mode (creates an orphan branch + git worktree and auto-commits transcripts when you run git commit):

cg install --agent claude --format html --branch transcripts

cg uninstall

cg uninstall --purge

cg uninstall --out .transcripts
