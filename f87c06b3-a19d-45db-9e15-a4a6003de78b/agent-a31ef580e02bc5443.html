Implementation Plan: Make --branch Optional in cg install

Overview

When --branch is omitted, cg install should operate in "simple directory mode": create a plain output directory, install the SessionEnd hook, and update .gitignore. No orphan branch, no worktree, no post-commit hook. When --branch is explicitly provided, the existing behavior is preserved.

A new --out flag (default ".transcripts") controls the output directory name.

1. Config Struct Changes (~/work/chitragupt/install/install.go)

Update the Config struct:

type Config struct {
    Agent   string   // agent name, e.g. "claude"
    Formats []string // output formats, e.g. ["html", "json"]
    Branch  string   // orphan branch name; empty means simple directory mode
    OutDir  string   // output directory name relative to repo root, e.g. ".transcripts"
    Dir     string   // git repository root (auto-detected if empty)
}
  • Branch changes from always-set to optional. Empty string means "no branch mode".
  • OutDir is new. Default ".transcripts". This is the directory name (not path) relative to repo root. The code will use filepath.Join(cfg.Dir, cfg.OutDir) as the absolute path.

2. Run() Function Changes (~/work/chitragupt/install/install.go)

Replace the fixed step list with conditional logic:

func Run(cfg Config) error {
    if cfg.Dir == "" {
        dir, err := gitRoot()
        if err != nil {
            return fmt.Errorf("not a git repository (run from inside a repo): %w", err)
        }
        cfg.Dir = dir
    }

    if cfg.OutDir == "" {
        cfg.OutDir = ".transcripts"
    }

    outPath := filepath.Join(cfg.Dir, cfg.OutDir)

    branchMode := cfg.Branch != ""

    if branchMode {
        // Branch mode: orphan branch + worktree
        if err := createOrphanBranch(cfg.Dir, cfg.Branch); err != nil {
            return fmt.Errorf("create orphan branch: %w", err)
        }
        if _, err := os.Stat(outPath); os.IsNotExist(err) {
            if err := addWorktree(cfg.Dir, cfg.Branch, outPath); err != nil {
                return fmt.Errorf("add git worktree: %w", err)
            }
        }
    } else {
        // Simple directory mode: just create the directory
        if err := os.MkdirAll(outPath, 0o755); err != nil {
            return fmt.Errorf("create output directory: %w", err)
        }
    }

    if err := ensureGitignore(cfg.Dir, cfg.OutDir); err != nil {
        return fmt.Errorf("update .gitignore: %w", err)
    }

    if err := installClaudeHook(cfg.Dir, cfg.Agent, cfg.Formats, cfg.OutDir); err != nil {
        return fmt.Errorf("install Claude Code hook: %w", err)
    }

    if branchMode {
        if err := installPostCommitHook(cfg.Dir, cfg.OutDir); err != nil {
            return fmt.Errorf("install git post-commit hook: %w", err)
        }
    }

    return nil
}

# cg-transcripts-start
# Auto-commit transcripts to the transcripts worktree.
# Installed by cg install.
REPO_ROOT="$(git rev-parse --show-toplevel)"
WORKTREE="$REPO_ROOT/%s"
if [ -d "$WORKTREE/.git" ] || [ -f "$WORKTREE/.git" ]; then
  cg index --dir "$WORKTREE" 2>/dev/null || true
  MAIN_SHA="$(git rev-parse --short HEAD)"
  unset GIT_DIR GIT_INDEX_FILE GIT_WORK_TREE
  git -C "$WORKTREE" add -A 2>/dev/null
  git -C "$WORKTREE" diff --cached --quiet 2>/dev/null || \

fi
# cg-transcripts-end

}

7. Potential Challenges

  • Backward compatibility: Users who previously ran cg install without --branch got branch mode (default was "transcripts"). After this change, running cg install without --branch gives simple mode. This is the intended behavioral change, but the command description should make the two modes clear.

  • Re-running install in a different mode: If a user first installed with --branch and then re-runs without it, or vice versa, the directory may already exist as a worktree or plain directory. The Run() function should handle this gracefully. For now, if the output directory already exists, skip creation in both modes (the current worktree check does if worktreeExists and skips). This is acceptable for an initial implementation.

  • The save-transcript.sh exit 0 on missing dir: Currently the hook exits silently if .transcripts/ does not exist. In simple directory mode the directory is a plain dir that could be deleted. Changing to mkdir -p is a small improvement. However, this changes behavior for existing branch-mode installs too. An alternative is to keep exit 0 -- the user can re-run cg install to recreate the directory. I'd recommend changing to mkdir -p since it makes the hook self-healing in both modes.
