Changing to `mkdir -p` is a small improvement. However, this changes behavior for existing branch-mode installs too. An alternative is to keep `exit 0` -- the user can re-run `cg install` to recreate the directory. I'd recommend changing to `mkdir -p` since it makes the hook self-healing in both modes.
